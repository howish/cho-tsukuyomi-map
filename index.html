このサイトについて

映画『超かぐや姫！』 オンリー同人即売会など、同人即売会の 非公式 fan guide を集めたハブサイト。各イベントごとに：

  • 📍 会場マップ + 全サークル一覧
  • 📕 各ブースの お品書き画像・X リンク・価格・注意事項
  • 🦊 CP / 🤝 合同 / 🔞 R-18 等のタグフィルター
  • ⭐ お気に入り登録 (localStorage)
  • 🔗 ブース ID deep-link (`//#A-04`)

評価ランキング・完売予測は意図的に含めない中立 fan guide ポリシーで運営。 各サークル情報は作家の X 公開ポストから引用、X リンクから原典確認推奨。

Built by @howish ／ ソースは GitHub ／ 同人即売会の guide 追加リクエストは Issue / PR で歓迎🌿
