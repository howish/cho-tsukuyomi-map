🔖 Booth Image Bookmarklet

1-click image extractor for FB / Threads / Plurk / X / Bluesky posts. Runs in your logged-in browser, so it sees what you see — bypasses the limited-render that the scrape pipeline hits for friends-only / low-engagement posts.

1. Drag this to your bookmarks bar

📸 Grab Booth Imgs

Mobile: tap-and-hold the link → add to bookmarks → edit the bookmark → strip the leading https://yachi8000.app/... wrapper if the browser added one. Desktop: just drag to the bookmark bar.

2. Usage

(a) Open the post you want to feed (FB share / Threads / Plurk / X status / Bluesky post).
(b) Click the 📸 Grab Booth Imgs bookmarklet.
(c) A modal pops up with the images it found + author/text/source URL. Press Copy.
(d) Paste the JSON into a new GitHub Issue (or future "/import" panel) — Yachiyo's pipeline downloads + R2-uploads.

Supported platforms

facebook.com threads.com / .net plurk.com x.com / twitter.com bsky.app

Output format

{
  "platform": "fb",
  "source_url": "https://www.facebook.com/yishi89/posts/pfbid02PkNo...",
  "author": "yishi89",
  "text": "【05/30 創集繪-百合ONLY T15 品書】...",
  "images": [
    "https://scontent.ftpe20-1.fna.fbcdn.net/v/t39.30808-6/.../705968441_.jpg",
    ...
  ]
}

Yachiyo can paste this directly into the apply pipeline — no scraper involvement needed for image discovery. The CDN URLs are signed but publicly downloadable for weeks, so the apply runner can pull them anonymously.

What it does (TL;DR)

  • FB: scoops every scontent.fbcdn.net/v/t39.30808-6/ image bigger than 300x300px (skips comment avatars + tiny thumbs), sorts by area
  • Threads: instagram.fna.fbcdn.net carousel images, prefers t51.82787-15 (post media) over t51.82787-19 (profile pics)
  • Plurk: images.plurk.com/<hash>.jpg sans mx_ thumbnails
  • X: pbs.twimg.com/media/ upgraded to ?name=orig
  • Bluesky: cdn.bsky.app/img/feed_fullsize/ only (skips avatars + quote-card thumbs)
